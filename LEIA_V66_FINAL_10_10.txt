YTALSEG ERP - V66 FINAL 10/10

Correções aplicadas:
1. Menu lateral com rolagem interna e espaçamento ajustado para não cortar opções abaixo de Diagnóstico.
2. Aba Relatórios com opção Salvar relatório editável.
3. Aba Relatórios com opção Abrir backup/relatório para carregar um JSON já salvo e continuar editando dias/serviços.
4. Aba Relatórios com opção Editar último relatório salvo no histórico local.
5. Valor da nota corrigido para não deformar cards/campos; fonte reduz automaticamente e não quebra layout.
6. Backend configurado para iniciar junto com o Electron ao abrir o .exe.
7. Backend incluído no pacote do instalador via extraResources.
8. PDF via Electron habilitado por IPC: salvarPDF.
9. Backup manual via Electron habilitado por IPC: gerarBackup.
10. Mantidos login, cadastro, relatórios, clientes e funções existentes.

Como testar sem gerar exe:
1. Abrir PowerShell na pasta frontend.
2. Rodar: npm install
3. Rodar: npm run desktop

Como gerar o instalador .exe:
1. Abrir PowerShell na pasta frontend.
2. Rodar: npm install
3. Rodar: npm run dist
4. O instalador sairá em: frontend\release

Observação:
Depois de instalado, o backend FastAPI sobe automaticamente junto com o aplicativo.
